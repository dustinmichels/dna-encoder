How does it work?

{% endblock %} {% block mainContent %}

Step 1. Get a text message to convert to DNA

Step 2. Take a single letter and look up its numerical code in an ASCII table

ASCII is a character encoding system that assigns a numerical value to standard english characters.

For example, the letter "lowercase h" is assigned the number 104.


Step 3. Convert the numerical value into binary

All ASCII characters can be encoded using 8 bits.

Step 4. Look at the first two bits, and convert them into a DNA base.

Two bits can represent four numbers.

Hey! There are four bases in DNA!

This means we can arbitrarily assign a number (0-3) to a base (A,C,G, or T.)

Step 5. Repeat for all eight bits

For a given character, the program currently builds the DNA sequence left to right, while reading bits from right to left.

So "h" gets encoded as AGGC rather than CGGA.

Step 6. Repeat this process for all the words in the message!
